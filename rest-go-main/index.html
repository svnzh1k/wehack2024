Restaurant API Documentation

HTTP Method Endpoint Description Authentication Request Body Response
POST /auth/signup Create a new user account No User details (JSON) {"message": "User signed up successfully", "user": { "id": 1, "username": "example", "role": "user", "bank": 0 }}
POST /auth/login Login an existing user No User credentials (JSON) {"token": "JWT token"}
POST /admin/menu Add item to menu Yes (Admin) {"token": "JWT token", "item": { "name": "ItemName", "price": 10 }} {"success": "Item added"}
DELETE /admin/menu/{id} Remove item from menu Yes (Admin) {"token": "JWT token"} {"success": "Item removed"}
PATCH /admin/{id} Update order status Yes (Admin) {"token": "JWT token"} http code 200 if success and 500 otherwise
POST /menu/{id} Place order Yes (User) {"token": "JWT token", "price": 10} {"error": "order placed successfully"}
GET /menu Show menu No [{"id": 1, "name": "ItemName", "price": 10}, {...}]
GET /admin/revenue Get revenue Yes (Admin) {"token": "JWT token"} {"items_sold": 10, "revenue": 100}
